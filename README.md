# ODA-IDE

This is a simply IDE for C and C++ development.

## Compile

Necessary to have qmake4 or qmake5
```bash
cd /desired/path/
mkdir ODA-IDE
git pull https://github.com/bartfaimate/ODA-IDE
qmake ODA-IDE.pro
make
```
