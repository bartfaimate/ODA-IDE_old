# ODA-IDE

This is a simply IDE for C and C++ development.

## Compile

Necessary to have qmake4 or qmake5
```bash
cd /desired/path/
git clone https://github.com/bartfaimate/ODA-IDE
mkdir build
cd build
qmake ../ODA-IDE.pro
make
```
